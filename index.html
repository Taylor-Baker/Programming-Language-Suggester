Question 1: Desktop or Mobile

One of the very first things you will need to consider when approaching code is what your intended use for that code will be. Are you creating advanced desktop apps with robust features? Or would the best medium for your particular code something more lightweight and mobile, possibly with less features but certainly easier to access. Perhaps you would like to create code that can run on both desktop mobile, communicating seamlessly between the two. Knowing what platform your code will be most useful on is a great first step to figuring out what coding language you should first learn.

That being said, are you more interested in creating desktop applications, mobile applications, or both?
