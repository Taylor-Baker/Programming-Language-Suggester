Question 1 : Consider Your Platform

One of the first things to consider when writing code is what platform will best meet the needs of your application. Are you creating advanced desktop apps with robust features, or would the best medium for your code be something more lightweight and mobile? Perhaps you would like to create code that can run on both desktop and mobile, communicating seamlessly between the two.

Knowing what platform your code will be most useful on is a great first step to figuring out what coding language you should learn first.
